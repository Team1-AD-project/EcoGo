# 优惠券系统快速启动指南

## 🚀 快速查看效果

### 方法1：Android应用查看

#### 启动步骤
1. **打开Android Studio**
2. **导入项目**: `android-app` 文件夹
3. **同步Gradle**: 等待依赖下载完成
4. **运行应用**: 点击Run按钮或按 Shift+F10

#### 导航到优惠券页面
```
启动应用
  ↓
登录/跳过引导页
  ↓
点击底部导航栏"Profile"
  ↓
点击"Redeem"按钮
  ↓
🎉 进入优惠券页面！
```

#### 查看内容
- **商品兑换Tab**: 27件装扮和实物商品
- **优惠券Tab**: 28张NUS校园优惠券
  - 🏪 商城：所有可兑换优惠券
  - 🎫 我的券包：已兑换的券
  - ✅ 已使用：历史记录
  - ⏰ 已过期：过期的券

## 🐛 常见问题

### Q1: 优惠券显示为空？
**A**: 确保已运行后端服务，或检查MockData是否正确加载。

### Q2: 图标显示不正确？
**A**: 某些设备可能不支持所有emoji，这是正常现象。

### Q3: 如何添加新优惠券？
**A**: 
- Android端: 编辑 `MockData.kt` 中的 `VOUCHERS` 列表
- Web端: 使用管理后台的"新建优惠券"功能

### Q4: 如何修改积分价格？
**A**: 
- Android端: 修改 `MockData.kt` 中对应优惠券的 `cost` 值
- Web端: 点击"编辑"按钮修改
